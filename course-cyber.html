About This Course

What This Course Covers

The Wolves Team Cyber Security Fundamentals course is a hands-on, lab-driven program designed for Arabic speakers who want to enter one of the fastest-growing and highest-paying fields in technology. You will learn to think like both an attacker and a defender — the dual mindset that defines a real security professional.

Every module is paired with a real virtual lab environment. You will run actual penetration tests, analyze real malware samples, configure live firewalls, and conduct full network reconnaissance using industry-standard tools including Kali Linux, Wireshark, Metasploit, Nmap, and Burp Suite. No simulations — only real tools, real systems.

By course completion, you will have the skills to pursue roles as a SOC Analyst, Penetration Tester, Security Engineer, or Ethical Hacker — and the knowledge to pass industry certifications including CompTIA Security+, CEH, and eJPT.

Curriculum

What You Will Learn

8 battle-tested modules covering every layer of modern cyber security — from foundations to advanced offensive and defensive techniques.

MODULE 01
Cyber Security Foundations
Build the mindset and vocabulary of a security professional. Understand the threat landscape, attack vectors, and defense fundamentals before touching a single tool.
  • CIA Triad & Security Models
  • Types of Hackers & Threats
  • Attack Lifecycle (Kill Chain)
  • Security Laws & Ethics
MODULE 02
Networking for Hackers
Security without networking is impossible. Master TCP/IP, DNS, HTTP, routing, subnetting, and packet analysis — the foundation of every attack and defense technique.
  • TCP/IP & OSI Model Deep Dive
  • Wireshark Packet Analysis
  • DNS, HTTP/S Internals
  • VPN & Proxy Techniques
MODULE 03
Linux & Kali Mastery
Become fluent in Linux — the operating system of every security professional. Master the Kali Linux environment, scripting, and file system for offensive operations.
  • Linux Command Line Fluency
  • Kali Linux Setup & Config
  • Bash Scripting for Security
  • File Permissions & Privilege
MODULE 04
Reconnaissance & OSINT
Every attack begins with information gathering. Learn passive and active reconnaissance using Nmap, Maltego, Shodan, and OSINT frameworks to profile targets legally.
  • Nmap Scanning Techniques
  • OSINT with Maltego & Shodan
  • Google Dorking
  • Footprinting & Enumeration
MODULE 05
Exploitation & Metasploit
Learn to identify and exploit vulnerabilities using Metasploit Framework, manual exploitation techniques, and CVE research against real lab targets.
  • Metasploit Framework Mastery
  • Buffer Overflow Basics
  • CVE Research & Exploitation
  • Post-Exploitation Techniques
MODULE 06
Web Application Security
The majority of attacks target web applications. Master OWASP Top 10 vulnerabilities including SQL Injection, XSS, CSRF, IDOR, and broken authentication with Burp Suite.
  • OWASP Top 10 In-Depth
  • Burp Suite Professional
  • SQL Injection Mastery
  • XSS & CSRF Exploitation
MODULE 07
Malware Analysis & Defense
Understand how malware works from the inside. Analyze real malware samples in a safe sandbox environment, reverse-engineer behaviors, and build detection rules.
  • Static & Dynamic Analysis
  • Sandbox Environments
  • Ransomware Mechanics
  • YARA Rule Writing
MODULE 08
SOC, Incident Response & Certs
Operate like a real SOC analyst. Learn SIEM tools, log analysis, incident response playbooks, and targeted preparation for Security+, CEH, and eJPT exams.
  • SIEM & Splunk Basics
  • Incident Response Playbooks
  • Security+ Exam Prep
  • Full CTF Challenge Lab

FAQ

Frequently Asked Questions

Do I need a programming background to join?
No programming experience is required. Module 3 covers Linux and Bash scripting from scratch. However, students who have basic familiarity with computers and networks will progress faster in the later modules. If you can use a computer comfortably, you can start this course.
Is everything I learn in this course legal?
Absolutely. Every technique is taught in a controlled lab environment using intentionally vulnerable machines. We cover ethics and legal boundaries in detail from the very first module. All practical assignments are conducted against your own lab systems or authorized platforms like HackTheBox and TryHackMe.
What hardware do I need for the labs?
A laptop or desktop with at least 8GB RAM and 50GB free storage is recommended so you can run virtual machines locally. If your machine can't handle VMs, we provide access to cloud-based lab environments for all exercises. A mid-range laptop from 2018 or newer will handle everything in this course.
Will this course prepare me for CEH or Security+ certification?
Yes. Module 8 is entirely dedicated to certification preparation for both CompTIA Security+ and the eJPT. The CEH content is woven throughout the penetration testing modules. Students who complete all 8 modules report passing Security+ on their first attempt consistently.
How realistic are the lab environments?
Extremely realistic. We use industry-standard tools — Kali Linux, Metasploit, Burp Suite, Wireshark, Nmap — against intentionally vulnerable machines that simulate real enterprise networks. The setups mirror what you'll encounter in actual penetration testing engagements and CTF competitions.
What happens after I finish the course?
You receive your Wolves Team certificate and remain a lifetime member of our security community. We provide ongoing support for job applications, bug bounty questions, and certification prep. Many graduates go on to freelance engagements, SOC roles, and competitive CTF teams — our Discord stays active for all alumni.
